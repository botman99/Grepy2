Grepy version 2.6.0



What's New:

Version 2.6.0 - January 4, 2026
  • Fix a crash bug with RichText handling.
Version 2.5.0 - December 15, 2024
  • Make performance faster.
Version 2.4.0 - July 9, 2024
  • Fix a major performance and memory usage issue introduced in 2.3.0 version.
Version 2.3.0 - July 6, 2024
  • Remove 'Everything' search feature (since Everything can lock up for a while after deleting lots of files).
Version 2.2.0 - August 22, 2020
  • Position or Center dialogs properly using StartPosition.
Version 2.1.2 - April 27, 2020
  • Fixed a bug when detecting if Everything has indexed the drive being searched.
Version 2.1.1 - April 19, 2020
  • Added the ability to change File Information (ListBox) font and Search Results (RichText) font.
Version 2.1.0 - March 29, 2020
  • Replaced default RichTextBox with BetterRichTextBox which does auto-selection better.
    (there's mouse doubleclick word select mode and Ctrl mouse doubleclick advanced select mode. See "Grepy Details".)
  • Added a View option to show or hide line numbers in the search results pane.
Version 2.0.9 - August 17, 2019
  • Added support for searching Microsoft UNC (Universal Naming Convention) file paths.
  • Make sure Everything is not currently indexing before trying to search with Everything.
  • Make sure Everything has indexed the drive being searched (so we don't use it to search drives not being indexed by Everything).
  • Change the background color of the selected file so it's easy to see what was selected after clicking in the search results Rich Text Box.
  • Fix an issue with no results returned when the "Folder To Search" was the root of the disk (i.e. "C:\").
  • Fix an issue displaying the message "No files found matching the file specification:" when not finding any files that match the filespec.
Version 2.0.8 - July 9, 2019
  • Fix some crashing bugs and fix tab positions in RichTextBox.
Version 2.0.7 - June 16, 2018
  • Fix dialogs so they center on the parent.
Version 2.0.6 - March 27, 2018
  • Made the "No Matches Found" dialog be TopMost.
Version 2.0.5 - January 9, 2018
  • Fix a bug with processing Mac type text files (LF with no CR).
Version 2.0.4 - August 28, 2017
  • Fix a bug where adjusting 'Before' or 'After' slider when no matches are displayed would cause a crash.
  • Added a "No Matches Found" dialog to display when no matches are found when searching for text.
Version 2.0.3 - August 15, 2017
  • Fix a bug with 'After' Preview not showing lines when match is within 5 lines of end of file.
Version 2.0.2 - July 13, 2017
  • Fix a bug with worker threads not stopping when search is cancelled.
Version 2.0.1 - July 2, 2017
  • Lower the priority of worker threads.
Version 2.0.0 - April 29, 2017
  • Initial version.
