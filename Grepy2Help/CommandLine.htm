Grepy2 CommandLine

Grepy2 supports the following command line arguments:

Grepy2.exe [switches] [SearchFolder] SearchString FileSpec [FileSpec...]
Where optional [switches] can be any of the following:

-r = Recurse folders
-cs = Use Case Sensitive searching
-noregex = Don't use regular expressions when searching

The optional SearchFolder can be specified if you want to search from a specific folder. If no SearchFolder is specified, Grepy will start searching from the current directory.

The SearchString is the string of text that you want to search for. Regular Expressions are enabled by default so if you are using wildcard characters in your search string, make sure the search string is a valid regular expression.

The FileSpec arguments are the file specification that you want to use when searching for the search string. Normal MS-DOS file specifications can be used here. Multiple file specifications can be provided on the command line.

Examples:
Grepy2 test *.*
(Searches for the string "test" in all files in the current directory.)
Grepy2 -r -cs Main *.cpp
(Does a case sensitive search for the string "Main" in all .cpp files in the current directory and all subdirectories.)
Grepy2 -r "C:\Projects" new *.h *.cpp
(Searches for the string "new" in all .h and .cpp files in the 'C:\Projects' directory and all subdirectories.)

NOTE: "Grepy2 -?" will output usage information to the console.
